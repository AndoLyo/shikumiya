## 1. Instagramアルゴリズム（2025-2026年）

### 1.1 アルゴリズムの基本構造

Instagramは単一のアルゴリズムではなく、**フィード・ストーリーズ・Explore・リール・検索**それぞれに独立したAIシステムが動いている。

### 1.2 最重要ランキングシグナル（Adam Mosseri公式発表）

1. **視聴時間（Watch Time）** — コンテンツの滞在時間。最初の3秒が特に重要
2. **いいね率（Likes per Reach）** — リーチに対するいいね数の比率
3. **DMシェア数** — 友人にDMで送られた回数（最重要視されている）

> Instagram責任者は「リーチを伸ばす最も重要なシグナルは『送信（シェア）』である」と明言。

### 1.3 Exploreページに載る条件

- **ポスト人気シグナル**がフィードやストーリーズより重視される
- いいね・コメント・シェア・保存が多数のユーザーから集まる投稿が優先
- ユーザーの過去の行動パターン（いいね、保存、コメント、シェア傾向）に基づいて表示

### 1.4 2025年の重要変更

- **「ビュー数」が中心指標**に変化（いいね数から）
- **フォロワー数に関係なくリールが公平に評価**される仕組みが強化
- **オリジナルコンテンツ優遇** — TikTok/YouTube Shortsの透かし入り動画は評価が低い
- **2024年12月にハッシュタグフォロー機能が廃止**

## 3. エンゲージメント戦略

### 3.1 他者への「コメント戦略」— 効果あり

**効果**: 毎日10-15分、ターゲットアカウントへの意味のあるコメントを残すことで信頼構築・可視性向上・リーチ拡大が期待できる。

**ルール:**

| ルール | 詳細 |
|--------|------|
| **4語以上のコメント** | アルゴリズムは絵文字のみのコメントをスパム判定する |
| **具体的な内容** | 「素敵ですね」ではなく「この光の表現がすごく好きです、何のツールで作りましたか？」 |
| **投稿前30分ルール** | 自分の投稿前に30分間、他者のコンテンツにコメントする |
| **相互コメント** | 継続的に互いのポストにコメントするとアルゴリズムが関連性を認識する |
| **自分のコメントへの返信** | 数時間以内に返信すると追加エンゲージメントが発生し、投稿が伸びる |

**コメント返信の効果**: Buffer調査で、コメントに返信することでエンゲージメントが**21%向上**。

### 3.2 初動の重要性

- **投稿後60分以内**に200-250ビュー＋強いコメントエンゲージメントがあるとアルゴリズムが推進
- **コメント対いいね比率**: いいねした人の5%がコメントすると「会話が成立している」と判定

### 3.3 ハッシュタグ戦略（2026年版）

**結論: まだ有効だが、役割は大きく変化。リーチ全体の10-20%程度の貢献。**

| 項目 | 推奨 |
|------|------|
| **個数** | 3-5個（公式推奨）。最大でも10個 |
| **種類** | 高い関連性のニッチタグのみ |
| **避けるべき** | 30個フル使用、汎用タグ（#art #love）、禁止ハッシュタグ |
| **役割** | カテゴリ分類・検索シグナル（リーチブーストではない） |

### 3.4 コラボ投稿

- **最大5人まで**コラボレーターを追加可能
- コラボ投稿は**通常の2倍のインプレッション**と**1.66倍のエンゲージメント**
- 4-5人のコラボレーターで**3.8倍のエンゲージメント率**
- フィード投稿、カルーセル、リールに対応（ストーリーズは不可）
- 小規模でエンゲージメントが高いアカウントとのコラボが大アカウントより効果的

### 3.5 ストーリーズインタラクション

- ポール・質問・スライダーなどのインタラクティブ要素で**エンゲージメント30%増**
- コラボ投稿をストーリーズでシェアして露出拡大

## 4. ターゲットアカウントの発見・エンゲージメント方法

### 4.1 手動で見つける方法

1. **ハッシュタグ検索**: #AIアート #プロンプトエンジニアリング 等で検索、アクティブなクリエイターをフォロー
2. **Explore経由**: AI関連コンテンツを積極的にいいね・保存して、Exploreをチューニング
3. **競合のフォロワー**: 類似アカウントのフォロワー一覧からターゲットを発見
4. **リール内検索**: AIアート関連のリールからクリエイターを発見

### 4.2 API経由で見つける方法

- **Business Discovery API**: 他のビジネス/クリエイターアカウントの基本情報（フォロワー数、投稿数、bio等）を取得可能
- **Hashtag Search API**: ハッシュタグで投稿を検索可能（ただし7日間で30ユニークハッシュタグの制限）

## 5. シャドウバン（リーチ制限）の原因と対策

### 5.1 シャドウバンとは

アカウントへの通知なしに、ハッシュタグ検索・Explore・非フォロワーへの表示が制限される状態。**リーチが40-90%低下**、**ハッシュタグリーチが最大99%減**、**新規フォロワー獲得が95%減**。

### 5.2 シャドウバンの原因

| 原因 | 詳細 |
|------|------|
| **禁止ハッシュタグの使用** | #followforfollow, #snapchat 等。意図せず使用しても全体が制限される |
| **ボット的行動** | 1時間に40件以上のコメント、大量フォロー/アンフォロー |
| **非公式自動化ツール** | Jarvee等の自動いいね・自動フォロー・自動コメントツール |
| **コミュニティガイドライン違反** | 著作権侵害、センシティブコンテンツ等 |
| **同一デバイス/IP複数アカウント** | 同じ端末で複数アカウントを運用するとフラグが立つ |
| **不自然な投稿パターン** | きっかり6時間おきなど、機械的な投稿間隔 |

### 5.3 回避ルール

1. **ハッシュタグは事前にチェック** — 禁止リストを確認してから使用
2. **1時間あたりのアクション上限を守る** — コメント30件以下、いいね60件以下を目安に
3. **非公式自動化ツールは絶対に使わない**
4. **投稿間隔にランダム性を持たせる** — 完全に等間隔にしない
5. **シャドウバン期間は7-21日** — その間は通常のオーガニック投稿のみにする

## 6. API自動化 vs 手動作業の完全整理

### 6.1 API（Graph API）で自動化できること

| 機能 | 制限 | 備考 |
|------|------|------|
| **投稿の公開（写真・カルーセル・リール）** | 24時間で50-100件 | スケジュール投稿も可能 |
| **自分の投稿へのコメント取得** | 200リクエスト/時間 | GET /{ig-media-id}/comments |
| **自分の投稿のコメントへの返信** | 200リクエスト/時間 | POST /{ig-comment-id}/replies |
| **コメントの非表示・削除** | 200リクエスト/時間 | モデレーション用 |
| **DM自動返信** | 200件/時間 | 24時間以内にエンゲージしたユーザーのみ |
| **ハッシュタグ検索** | 7日で30ユニークハッシュタグ | 投稿一覧の取得 |
| **Business Discovery** | 200リクエスト/時間 | 他のビジネスアカウントの基本情報 |
| **インサイト取得** | 200リクエスト/時間 | フォロワー数推移、リーチ等 |

### 6.2 APIでは絶対にできないこと（手動のみ）

| 機能 | 理由 |
|------|------|
| **他者の投稿にいいね** | APIエンドポイント自体が存在しない |
| **他者の投稿にコメント** | ハッシュタグ経由で発見した投稿へのコメントは明示的に禁止 |
| **フォロー/アンフォロー** | APIエンドポイントなし |
| **ストーリーズの閲覧** | APIエンドポイントなし |
| **ストーリーズへの返信** | APIエンドポイントなし |
| **他者のフォロワーリスト取得** | プライバシー制限 |
| **Exploreページの閲覧** | APIエンドポイントなし |
| **コラボ投稿の作成** | APIエンドポイントなし |

## 8. 重要ポイントまとめ

### やるべきこと
1. **リール中心**のコンテンツ戦略（60-70%）
2. **DMシェアされるコンテンツ**を意識して設計
3. **毎日15分**のターゲットアカウントへの手動コメント
4. **カルーセルで保存を狙う**教育コンテンツ
5. **ハッシュタグは3-5個**、高関連性のニッチタグのみ
6. **コラボ投稿**を積極的に活用
7. **API自動化は投稿公開・コメント返信・分析**に限定

### 絶対にやってはいけないこと
1. **自動いいね・自動フォロー・自動コメントツール**の使用
2. **1時間に40件以上のコメント**
3. **禁止ハッシュタグ**の使用
4. **ハッシュタグ30個フル使用**
5. **完全に等間隔の機械的投稿**
6. **非公式APIの使用**
7. **他者の投稿への自動コメント**（API経由での試みは明示的にブロック）

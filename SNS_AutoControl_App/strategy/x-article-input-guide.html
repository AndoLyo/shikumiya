1. X記事エディタでできること一覧

X記事エディタは、普通のツイート（ポスト）とは違い、文字を太くしたり、斜めにしたり、打ち消したりできます。使える装飾の一覧がこちらです。

装飾の名前どういう見た目になるか対応？
太字（ボールド）文字が太くなる✅ 対応
斜体（イタリック）文字が斜めになる✅ 対応
下線（アンダーライン）文字の下に線が引かれる✅ 対応
打ち消し線文字の上に横線が引かれる（こんな感じ）✅ 対応
見出し大きく太い文字になる（セクションのタイトル向き）✅ 対応
小見出し見出しより少し小さい太字✅ 対応
引用ブロック左に青い縦線がついた引用風の表示✅ 対応
箇条書き（●）先頭に「●」がつくリスト表示✅ 対応
番号つきリスト先頭に「1. 2. 3.」がつくリスト表示✅ 対応
リンク（URL）タップするとWebページに飛ぶ青い文字✅ 対応
コード表示プログラムのソースコードを表示する枠❌ 非対応
表（テーブル）エクセルのような格子状の表❌ 非対応

2. 方法① ツールバー（画面上部のボタン）で入力

記事エディタの上部にあるアイコンが並んだバーを使う方法です。マウスでボタンを押すだけなので、一番わかりやすい方法です。

🔤 文字の装飾ボタン（左側）

ボタンの見た目機能使い方
B太字にする①ボタンを押す → ②文字を打つ → ③もう一度ボタンを押して解除
I斜体にする同上
S打ち消し線同上

📝 ブロック種類の切り替え（ドロップダウン）

ツールバーの「本文 ▼」と書かれた部分をクリックすると、メニューが開きます。

選択肢どうなるか
見出しカーソルがある行が大きな見出し文字になる
小見出し見出しより一段小さい見出し文字になる
本文通常の文字に戻る
💡 ポイント：ドロップダウンは「その行全体」のスタイルを変えます。行の一部分だけを見出しにすることはできません。

📎 その他のボタン（右側）

ボタン機能使い方
❝（引用マーク）引用ブロックにする押すと行の左に青いバーが表示される
● リスト箇条書きにする押すと行頭に「●」がつく
1. リスト番号つきリストにする押すと行頭に番号がつく
🔗 リンクリンクを挿入する押すとURL入力画面が出る
😊 絵文字絵文字を選ぶ押すと絵文字一覧が出る
🖼 メディア画像・動画を挿入するファイルをアップロードする画面が出る

3. 方法② キーボードショートカットで入力

キーボードの複数のキーを同時押しするだけで装飾できます。マウスを使わず高速に入力できるので、慣れるとこれが一番早いです。

やりたいことキー操作（Windowsの場合）覚え方のヒント
太字にするCtrl + BBはBold（太字）の頭文字
斜体にするCtrl + IIはItalic（斜体）の頭文字
下線を引くCtrl + UUはUnderline（下線）の頭文字
打ち消し線Ctrl + Shift + XXは×（バツ）のイメージ
リンクを入れるCtrl + KKはlinK（リンク）のK
箇条書きにするCtrl + Shift + 7—
番号つきリストにするCtrl + Shift + 8—
引用ブロックにするCtrl + Shift + 9—
文字を大きくする（見出し方向）Ctrl + Shift + .「.」は右矢印（大きくなる方向）
文字を小さくする（本文方向）Ctrl + Shift + ,「,」は左矢印（小さくなる方向）
💡 使い方：①装飾したい文字を打つ前にキーを押す → ②文字を打つ → ③同じキーをもう一度押して解除。
すでに打った文字を装飾する場合は、文字をドラッグ選択してからキーを押してもOKです。

4. 方法③ 記号を打って自動変換（Markdown風）

行の先頭で特定の記号を打ってスペースを押すと、自動的にその行が変換されます。ブログ記事を書くときによく使われる「Markdown」という書き方に似ています。

行の先頭に打つ記号スペースを押すと…補足
# （シャープ＋スペース）→ 見出しに変わる「# 」を打った瞬間に記号は消えて見出しブロックに変換
## （シャープ2つ＋スペース）→ 小見出しに変わる同上
- （ハイフン＋スペース）→ 箇条書きリストに変わる+ や* でもOK
1. （数字＋ピリオド＋スペース）→ 番号つきリストに変わる2)の形でもOK
> （大なり記号＋スペース）→ 引用ブロックに変わる>> でもOK
⚠️ 注意：この方法は「本文」モードの空行でのみ動作します。すでに見出しモードになっている行では記号がそのまま文字として入力されます。

5. 方法④ 書式つきテキストをコピペで入力

他のWebページやメモ帳アプリなどで装飾つきのテキストをコピーして、X記事エディタにCtrl+Vで貼り付ける方法です。

貼り付けで反映されるもの・されないもの

コピー元の装飾貼り付け後どうなるか
太字✅ そのまま太字で反映
斜体✅ そのまま斜体で反映
打ち消し線✅ そのまま打ち消し線で反映
リンク（青い文字）✅ リンクとして反映
下線⚠️ 下線は消えて普通の文字になる
コード表示❌ 普通の文字になる（非対応）
見出し（大きい文字）⚠️ 大きさは反映されず普通の文字になる場合あり
💡 おすすめのコピー元

GoogleドキュメントやNotionなどで書式つきテキストを作成し、そこからコピーしてX記事に貼り付けると、太字・斜体・打ち消し線・リンクがまとめて反映されて便利です。

6. 方法⑤ JavaScript（プログラム）で入力

ブラウザの開発者ツール（F12キーで開くコンソール）を使い、プログラムで直接エディタに書式つきテキストを入れる方法です。自動投稿ツールを作りたい方向けです。

方法A: HTML形式でペーストする（一番おすすめ）

クリップボードに「HTML形式の書式データ」をセットして貼り付ける方法です。太字・斜体・打ち消し線・リンクがすべて一度に反映されます。

// エディタ（文章を入力するエリア）を取得
const editor = document.querySelector('[contenteditable="true"]');

// 貼り付けたいHTMLを用意
const html = '<p><b>太字</b> <i>斜体</i> <s>打ち消し線</s> <a href="https://x.com">リンク</a></p>';

// クリップボードデータを作成
const clipboardData = new DataTransfer();
clipboardData.setData('text/html', html);
clipboardData.setData('text/plain', '太字 斜体 打ち消し線 リンク');

// 貼り付けイベントを実行
const pasteEvent = new ClipboardEvent('paste', {
  bubbles: true,
  cancelable: true,
  clipboardData: clipboardData
});
editor.dispatchEvent(pasteEvent);

方法B: insertHTMLコマンドを使う

もっと短いコードで入力する方法です。ただし、対応する装飾が限られます。

// 太字・斜体・打ち消し線はOK。下線やコードは反映されない。
document.execCommand('insertHTML', false,
  '<b>太字</b> <i>斜体</i> <s>打ち消し線</s>'
);

方法AとBの比較

方法A（ClipboardEvent）方法B（insertHTML）
太字✅✅
斜体✅✅
打ち消し線✅✅
リンク✅⚠️ 場合による
下線❌❌
コード量やや多い少ない
おすすめ度⭐⭐⭐⭐⭐

7. 対応していないもの（入れても無視されるもの）

以下の装飾は、どの方法で入力してもX記事エディタでは反映されません。

装飾状況
コード表示（<code>）プレーンテキスト（普通の文字）になる
表（テーブル）エディタに機能がない
文字色の変更エディタに機能がない
文字サイズの自由指定見出し／小見出し／本文の3段階のみ
画像のインライン表示画像は専用の挿入ボタンからのみ
💡 HTML貼り付けで<u>（下線）タグを入れても反映されませんが、ショートカット（Ctrl+U）なら下線が引けます。つまり下線は「ペーストでは入らないが、手動操作では使える」特殊な装飾です。

8. 結論：おすすめの入力方法は？

🏆 手動で入力する場合

場面おすすめの方法
太字・斜体・打ち消し線キーボードショートカット（Ctrl+B / Ctrl+I / Ctrl+Shift+X）が最速
見出し・小見出し行頭で# （シャープ＋スペース）が最速
箇条書き行頭で- （ハイフン＋スペース）が最速
引用ブロック行頭で> （大なり＋スペース）が最速
リンク挿入Ctrl+KでURLと表示テキストを入力
複数の装飾をまとめて入力Googleドキュメント等で書式つきテキストを作りCtrl+Vで貼り付け

🤖 プログラムで自動入力する場合

ClipboardEvent + text/htmlを使った貼り付けが、最も多くの書式を一度に反映できて確実です。

使えるHTMLタグ：<b> <i> <s> <a href="...">

📋 すぐ使えるHTMLテンプレート（コピペ用）
<b>太字にしたい文字</b>
<i>斜体にしたい文字</i>
<s>打ち消したい文字</s>
<a href="https://example.com">リンクの表示テキスト</a>

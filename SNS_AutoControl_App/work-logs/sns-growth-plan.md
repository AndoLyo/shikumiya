# SNS成長プラン（2026-03-24策定）

## 1. SEO対策 — noteタイトル改善（完了）

**ターゲット:** Claude Codeを使っているが使い勝手がイマイチわかっていない人
**前提:** X→note流入は減少傾向。Google検索からの流入がメインになる

### タイトル改善済み（#2〜#7）

| # | 新タイトル |
|---|----------|
| #2 | Claude Codeの使い方——AIエージェント18体をコードなしで作った方法 |
| #3 | Claude Codeスキルを自動で成長させる方法——使うほど賢くなる仕組み |
| #4 | 【テンプレ配布】Claude Codeスキルの作り方——コピペで使える設計図 |
| #5 | Claude Codeをスマホから操作する方法——Discord Bot+GASで外出先でも開発 |
| #6 | Claude CodeのAIエージェントが矛盾したときの直し方——CLAUDE.mdルール設計 |
| #7 | Claude CodeでSNS投稿を自動化する方法——テーマ1つでX・IG・画像を一括生成 |

**方針:**
- 全タイトルに「Claude Code」を含める（検索の起点）
- 「〜した話」→「〜する方法」「〜の作り方」（How-to検索を拾う）
- 「使い方」「作り方」で初心者〜中級者の検索にヒットさせる

## 2. X投稿頻度 — 2日に1回ペース

**現状:** しばらく休止 → 今週から再開、2日に1回ペース
**目標:** 週4投稿を定着させる

### 曜日別ネタの型

| 投稿日 | ネタの型 | 例 |
|--------|---------|-----|
| **月** | 今週やること宣言 | 「今週はSNS自動投稿の○○機能を実装する」 |
| **水** | 開発中の気づき（メモ感覚） | 「IG APIのレート制限、意外とここがハマる」 |
| **金** | 今週の成果物（スクショ or 動画） | 実装画面のキャプチャ、動作デモ |
| **日** | トレンド・感想 or 来週の予告 | AI画像生成の新機能レビュー |

### 界隈外へのリーチ

同じネタでも届ける相手を変えて切り口を変える：

| 界隈 | 切り口 | 例 |
|------|--------|-----|
| **個人事業主・フリーランス** | AIで業務を自動化した実例 | 「個人事業のSNS運用、全部AIに任せてみた」 |
| **ノーコード勢** | コード書かずにAIエージェント作れる | 「マークダウンだけでAIチーム18体作った方法」 |
| **note発信者** | noteのPV・収益を伸ばす仕組み | 「記事1本からSNS投稿を全自動生成する仕組み」 |

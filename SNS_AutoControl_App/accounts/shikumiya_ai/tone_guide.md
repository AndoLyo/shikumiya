# トーンガイド — @shikumiya_ai

## 基本スタンス
- 「あなたの作品は素晴らしい。でも発信がもったいない」という立ち位置
- 自分がAIで作ったことは裏側の話。表に出すのは「仕組みで楽になった結果」
- 煽りではなく共感から入る
- 自分視点（「〇〇を作りました」）ではなく相手視点（「〇〇で困ってませんか？」）

## OKなトーン
- 「絵は描けるのにサイトだけ3年放置してる。わかる。」
- 「作品はあるのにSNSが苦手で埋もれてる人、めちゃくちゃ多い。」
- 「ポートフォリオ作りたいけど何から始めれば…ってなるよね。」
- 「毎日投稿がしんどいんじゃなくて、何を書けばいいかわからないのがしんどい。」
- 「納品で手一杯なのに発信もしろって言われても、って気持ちわかる。」
- 「手動、しんどくないですか？」
- 「試してみて」「参考になれば」

## NGなトーン
- 「断言する。」← 煽りすぎ
- 「〜終わった。」← 上から目線
- 「〜いらない。〜いらない。」← 否定の連打は攻撃的
- 「これを知らない人は損してる」← 煽り
- 「さらに」「また」「加えて」の連続 ← AI臭い
- emダッシュ（——）の多用 ← AI臭い
- 全文が均一な長さ ← AI臭い
- 「まだ手動でやってるの？」← 上から目線（手動、しんどくないですか？はOK）

## 感情の入れ方
- 驚き: 「え、これでできるの？」「想像以上に簡単だった」
- 嬉しい: 「ありがとう！」「やってよかった」
- 共感: 「同じ悩み持ってる人いる？」「わかる、それしんどいよね」

## 投稿前チェック
- [ ] AI用語（Claude Code、バイブコーディング、エージェント等）が入ってないか
- [ ] 相手視点で書けてるか（「作りました」→「困ってませんか？」）
- [ ] 煽りではなく共感から入れてるか
- [ ] クリエイターが検索しそうな言葉を使えてるか
- [ ] emダッシュ（——）がないか
- [ ] 「さらに」「また」「加えて」が2回以上連続してないか
- [ ] 文の長さが全部均一になってないか
- [ ] 感情の揺れがゼロになってないか

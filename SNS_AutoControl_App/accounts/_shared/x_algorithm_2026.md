# Xアルゴリズム戦略メモ（2026年4月）

## 核心
- **リプライ往復 = 75-150x**（最強シグナル）
- **いいね = 1x**（最弱。廃止済み）
- **外部リンク = 最大40%減点**
- **滞在時間（Unregretted User Seconds）が最重要指標**
- **量 < 質**（1日5投稿 < 質の高い1投稿）

## 導線設計
- 本文にリンク貼らない
- 「リプから」で自己リプライに誘導
- 自己リプライにnoteリンク

## アカウント評価（TweepCred）
- +65以上でアルゴリズム優遇
- フォロー少 / フォロワー多 = 高評価
- 毎日最低1投稿で評価維持
- Premium加入でスコア2-4倍

## X Articles
- API非対応（手動 or Selenium）
- noteの代替としてX内で完結
- 滞在時間を稼げる
- 記事末尾に1箇所だけリンク可

## 投稿設計チェックリスト
- [ ] 1行目に結論があるか
- [ ] 自分の実体験・独自データが含まれるか
- [ ] 画像または動画が添付されているか
- [ ] 外部リンクが本文に入っていないか
- [ ] ハッシュタグを使っていないか
- [ ] AI画像生成のジャンルから外れていないか
- [ ] リプライしたくなる問いかけが含まれているか

## リプライのルール
### 禁止（無価値リプライ → シャドウバンリスク）
- 「すごいですね！」「いいですね！」「同意です！」
- コピペリプライ（同一テキスト繰り返し）
- 絵文字のみ
- 自分のリンクを貼る

### 必須（価値提供型）
- 自分の体験・データを含める
- 独自の視点を加える
- 2-3文で簡潔に
- 投稿から2時間以内のフレッシュな投稿に絞る
- 開始フレーズを毎回変える（パターン検出回避）

### リプライ例（OK）
```
これ、自分もやってみたんですが、○○の部分で△△という結果になりました。
□□を組み合わせるとさらに効果的かもしれませんね。
```

## 引用RTのルール
- いいね10件以上の投稿のみ対象（低インプレ投稿は引用しない）
- 自分の意見・体験を必ず追加（140字以内）
- 1日最大5件（やりすぎはスパム判定）

## フォローのルール
- ターゲットDBから自動選定
- 1日100件以下（安全マージン）
- 同じユーザーに1日1回まで
- フォロー/フォロワー比率 > 1.0 を維持

## シャドウバン回避
### 絶対NG
- 大量フォロー/アンフォロー（最大リスク）
- 同一テキスト繰り返し
- 1時間に100いいね超
- 1時間に30リプライ超
- IPアドレス頻繁切り替え

### 安全策
- undetected-chromedriver + 専用プロファイル使用
- アクション間にランダム待機（15-90秒）
- 日次上限に安全マージン（公式の80%以下）
- 毎回異なるテキスト（Claude CLIで毎回生成）

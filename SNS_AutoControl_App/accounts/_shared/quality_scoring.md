# 品質スコアリング基準（ライター自己採点用）

## 10項目評価（各1〜10点）

### 1. フックの強さ
- 1行目で「続きが読みたい」と思わせるか
- 10点: 思わずスクロールを止める / 1点: 素通りされる

### 2. 有益性
- 読者が「役に立った」と感じるか
- 10点: 即実践できる情報 / 1点: 既知の情報の繰り返し

### 3. 具体性
- 抽象論ではなく具体例・数字・手順があるか
- 10点: 「3分で」「5ステップで」等 / 1点: 「簡単に」「すぐに」等

### 4. テンポ感
- 文が短く歯切れよいか。読み疲れしないか
- 10点: リズムよく一気に読める / 1点: だらだら長い

### 5. 感情の揺れ
- 驚き・共感・困惑など感情が動くポイントがあるか
- 10点: 「え、マジ？」が複数ある / 1点: 淡々と情報が並ぶだけ

### 6. オリジナリティ
- 他の人が同じことを言ってないか
- 10点: Lyoだからこそ書ける体験 / 1点: 誰でも言える一般論

### 7. CTA適切さ
- 行動喚起が自然か。押し売りになってないか
- 10点: 読者が自然に次のアクションを取る / 1点: 唐突な宣伝

### 8. プラットフォーム最適化
- そのSNSの文化・文字数・形式に合ってるか
- 10点: そのSNSネイティブに見える / 1点: 別SNSのコピペ感

### 9. ターゲット適合
- 想定読者に刺さる言葉遣い・内容か
- 10点: 「これ自分のことだ」と思わせる / 1点: 誰に向けてるか不明

### 10. AI臭さの排除
- emダッシュ、「さらに」連続、均一な文長がないか
- 10点: 人間が書いたように見える / 1点: 明らかにAI生成感

## スコアリングルール

- **合格ライン: 平均7.0以上**
- 7.0未満 → 書き直し（最大2回）
- 2回書き直してもダメ → 棄却（捨てる）
- 個別項目が3点以下 → その項目だけ修正指示

## 類似度チェック

- 過去100件の投稿と比較
- 類似度0.85以上 → 自動棄却
- チェック対象: 1行目 + 本文の主要フレーズ
- 同じネタのループを防ぐ（転職エージェント30回事件の教訓）

## 棄却理由の記録

棄却した投稿は以下を記録:
```
{
  "draft_id": "xxx",
  "score": 6.2,
  "lowest_item": "フックの強さ: 3",
  "reason": "1行目が弱い。具体性がない",
  "similar_to": "post_078（類似度0.87）"
}
```
→ アナリストが傾向分析に使う

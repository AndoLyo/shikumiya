# プラットフォーム別 投稿ルール（全アカウント共通）

## X（Twitter）通常ポスト
- 140文字以内推奨
- ハッシュタグ: 1〜2個、文中に自然に配置
- CTA: 問いかけで終わる（「みんなどうしてる？」）
- パターン選択: 体験談ベースの共感型をデフォルト選択
- 初動30分が重要。リポスト = いいねの20倍の価値

## X 長文ポスト（8ブロック構成・必須）
1. **フック** — 冒頭3行で止める（問いかけ or 具体シーン描写）
2. **回答 → 感情 → ただし** — 答え→素直な反応→正直な注意喚起
3. **共感＋約束** — 「自分も最初〇〇だった」→「この通りやれば完了」
4. **超噛み砕いた説明** — 一言化 + ASCII図 + 言い換え
5. **ステップバイステップ** — 番号付き手順 + ⚠️ハマりポイント
6. **トラブルシューティング** — よくある失敗3つ
7. **実例（やってみた結果）** — 具体的な数字・出力・変化
8. **まとめ → 感情 → CTA** — 要約 + 質問形で閉じる or noteリンク

### テーマ別調整
- ハウツー系 → Block 5を厚く
- 体験談系 → Block 7を厚く
- 比較・検証系 → Block 4で図示 + Block 7で数字

### フォーマット
- Block間に `---` 区切り線
- 見出しは `**太字**`
- 1文20字以内
- 空行多め（スマホ読みやすさ重視）
- ハッシュタグは末尾に1〜2個

## X スレッド
- 5〜7ツイートのストーリー型（体験談ベース）
- リスト型より体験談型を優先
- 最後のツイートにnoteリンク + 「試してみて」

## Instagram
- **カルーセルがデフォルト**（リーチ1.4倍）
- 単体画像は作品紹介のときだけ
- ハッシュタグ: 5〜10個（末尾配置）
  - Big(1M+): 1〜2個
  - Mid(10k〜1M): 3〜5個
  - Niche(<10k): 2〜3個
- キャプション: 冒頭3行のみ表示（残りは「もっと見る」の下）
- CTA: 「保存しておくと便利」（フォローよりも保存重視）
- ピーク: 火・木 20〜21時

## Threads
- 最大500文字（最適100〜300文字）
- ハッシュタグ: 0〜1個（基本つけない）
- **問いかけ型をデフォルト選択**（コメント誘導が最重要）
- CTA: 「同じこと困ってる人いる？」
- Xと同じ文面は禁止。同じテーマでも切り口を変える
  - X: How-to型（「○○+△△で解決」）
  - Threads: 質問型（「○○、困ってない？」）

## 投稿スケジュール
| タイミング | 何を出す |
|---|---|
| 木曜 20時 | メイン投稿（エンゲージメント最大） |
| 金曜 12時 | X通常ポスト（リマインド） |
| 金曜 20時 | 記事告知 |
| 日曜 14時 | Xスレッド（じっくり読む層） |

### 全SNS同日投稿の順序
1. X長文（最初）
2. 1時間後 → Threads
3. 1.5時間後 → Instagram

## 画像ルール
- ちびキャラは「嬉しそう」「ワクワク」がデフォルト
- 困り顔は問題提起パートの画像だけ
- テキストは必ず入れる（ターゲット文+メインメッセージ）
- 背景はパステル系（ダーク背景はX用のみ許容）

## 文章の基本ルール
- 1文は20字以内を目安。25字を超えたら分割
- 「〜が、〜で、〜ため、〜だが」でつなげない。文を切る
- 修飾語・前置きは最小限。結論から書く
- 「結果→検証→やり方」の順で書く

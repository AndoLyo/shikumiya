# 投稿パターン一覧（ライターエージェントが使う型）

## X通常ポスト（6パターン）

### 1. 体験報告型（デフォルト）
「〇〇してみたらこうなった。」
→ 結果 + 感想 + 問いかけ

### 2. 共感→解決型
「〇〇で困ってない？」
→ 共感 + 解決策を1行 + 詳細はnoteリンク

### 3. Before/After型
「Before: 〇〇 → After: △△」
→ 変化を具体的に見せる + 数字があれば入れる

### 4. Tips型
「知ってた？〇〇で△△できる。」
→ 短く有益な情報1つ + 補足1行

### 5. 数字インパクト型
「〇〇を△△にしたら、□□が×倍になった。」
→ 具体的な数字で引きつける

### 6. 問いかけ型
「〇〇と△△、どっち使ってる？」
→ 選択肢を投げて反応を得る

## Threads（3パターン）

### 1. 質問型（デフォルト）
「〇〇、困ってない？」
→ 短い問いかけ + 自分の体験1行 + 「同じ人いる？」

### 2. 気づき共有型
「最近気づいたんだけど、〇〇って△△だよね。」
→ 発見 + 理由 + 「どう思う？」

### 3. あるある型
「〇〇あるある: △△」
→ 共感ネタ + 解決策ちょい見せ

## Instagram キャプション構造

### カルーセル用（デフォルト）
```
[1行目: フック（3行以内に収める — これだけが表示される）]
[空行]
[2行目〜: 本文（改行多め、読みやすく）]
[空行]
[まとめ or CTA]
[空行]
保存しておくと後で見返せます📌
[空行]
#ハッシュタグ1 #ハッシュタグ2 ... （5〜10個）
```

## 直近パターン重複回避ルール
- 直近3件と同じパターンは使用禁止
- 同じテーマが3日連続したらテーマツリーの別ブランチへ
- 類似度0.85以上の投稿は自動棄却

## パターンローテーション
ライターは投稿生成時に以下を守る:
1. 直近3件で使ったパターン名を確認
2. それ以外のパターンから選択
3. 選んだパターン名を投稿メタデータに記録

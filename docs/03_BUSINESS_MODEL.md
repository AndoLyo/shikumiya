## 1. 価格体系

### 1.1 プラン一覧

`[汎用]` 制作費0円+月額サブスクの3段階構造。初期費用ゼロで始められる安心設計。

| プラン | 価格 | 位置づけ |
|---|---|---|
| ライト | ¥3,000/月 | 入口。最低限のWeb存在感を確保 |
| ミドル | ¥8,000/月 | 本命。収益の柱。集客に効く機能を網羅 |
| プレミアム | ¥15,000~/月 | 高単価。AI・予約・採用など高機能 |

**制作費: 0円（全プラン共通）**

### 1.2 各プランの詳細

`[固有]` ターゲットは建築業（工務店・リフォーム・外壁塗装等）。

**ライト（¥3,000/月）**
- テンプレート選択によるサイト構築
- 施工写真10枚
- 会社概要
- 問い合わせフォーム
- 電話タップ（スマホワンタップ発信）
- SSL対応
- レスポンシブ対応
- 独自ドメイン対応
- 月1回更新

**ミドル（¥8,000/月）**
- ライトの全機能 +
- 施工実績詳細（Before/After）
- お客様の声
- ブログ機能
- Google Maps埋め込み
- SEO強化（JSON-LD / OGP）
- 月3回更新

**プレミアム（¥15,000~/月）**
- ミドルの全機能 +
- AIチャットボット
- 予約システム
- 採用ページ
- 多言語対応
- 360°ビューア
- PDF資料ダウンロード
- 動画埋め込み
- LINE通知連携

### 1.3 独自ドメイン

`[固有]` 全プラン対応。顧客がドメイン代を負担（年約1,400円〜）。Vercelカスタムドメイン設定で対応。

## 2. 解約条件

`[汎用]` 「いつでも解約OK」方式。縛りなしで信頼を得る。

| 項目 | 内容 |
|---|---|
| 解約タイミング | いつでも可能（縛りなし） |
| サイト | 解約月末まで公開 |
| 再開 | いつでも可能 |

`[判断記録]` 解約月末でサイト停止。「消さない」方式も検討したが、月額モデルでは非公開にしないと継続動機がなくなる。再開はいつでもOKにすることで心理的ハードルを下げる。

## 3. 収益シミュレーション

### 3.1 コスト構造

`[固有]`

```
固定費:
  Vercel Pro:    $20/月（約¥3,000）
  Claude API:    変動（¥30,000/月想定）
  合計:          約¥33,000/月

変動費:
  Stripe手数料:  3.6%/取引
```

### 3.2 損益分岐

`[汎用]` 最安プランだけでも12人で黒字化する低リスク構造。

```
ライトのみ:    ¥3,000 × 12人 = ¥36,000 > ¥33,000 → 黒字
ミドルのみ:    ¥8,000 × 5人  = ¥40,000 > ¥33,000 → 黒字
```

### 3.3 ターゲットMRR

```
目標: ミドル会員20人 = ¥160,000/月
```

### 3.4 12ヶ月シミュレーション（保守的）

```
前提: ライト月2人新規、ミドル月2人新規、解約率5%

月    ライト   ミドル   MRR
 1      2       2     ¥22,000
 3      6       6     ¥66,000
 6     11      11     ¥121,000
12     20      20     ¥220,000
```

### 3.5 スケーラビリティ

`[判断記録]` AIが編集・更新を処理するため人的天井はない。ボトルネックはVercel帯域（1TB/月）とClaude APIコストのみ。ミドル・プレミアムの比率が上がるほど利益率が改善する構造。

## 4. note記事（認知チャネル）

`[固有]` noteはテンプレート販売チャネルではなく、認知・信頼構築のチャネルとして活用。

| コンテンツ | 価格 | 目的 |
|---|---|---|
| 開発日記 | 無料 | SEO・信頼構築・制作過程の公開 |
| プロンプト集 | ¥500〜¥1,500 | AIアート層のファン化・収益 |
| クリエイター向け解説 | 無料 | ターゲット層Dへのリーチ |

`[判断記録]` テンプレート個別販売記事（¥980）は0購入で失敗。noteでの販売はプロンプト集など情報商材に限定し、サイト制作サービスはSaaS月額モデルに一本化。


# 06. カリキュラム設計

> 最終更新: 2026-04-02
> ステータス: 骨子確定（中身は後日作成）

---

## 1. カリキュラム概要

`[汎用]` この6モジュール構造は「サイトを作る→育てる→収益化する」の普遍的な流れ。ターゲットやサービスを変えても構造は同じ。

### 目的
1. ¥2,980会員への価値提供（解約防止）
2. ¥980→¥2,980アップグレードの動機（「ここからは会員限定」）
3. Phase 2コンサルの教材原型
4. noteに一部無料公開して集客

### 配信方法
- 自社サイト内の会員専用ページ（かわいさん方式: 静的HTML辞書型）
- Module 1は無料公開（集客用）
- Module 2以降は¥2,980会員限定

---

## 2. モジュール構成

### Module 1: サイト立ち上げ編（無料公開）

`[汎用]` サイト完成直後にやるべきこと。放置させない。

| レッスン | タイトル | 内容 |
|---|---|---|
| 1-1 | サイト完成おめでとう | 完成したサイトの確認方法。URL・表示チェック |
| 1-2 | SNSにURLを載せよう | X/Instagram/pixivのプロフィール設定方法 |
| 1-3 | 最初の3つの作品を選ぶ | どの作品を載せるか。選び方の基準 |
| 1-4 | 自己紹介を書く | 何を書くか。テンプレート付き |

`[固有]` AIアーティスト向けの具体例: pixivのリンク設定、X プロフカードへの反映方法

### Module 2: 作品の見せ方編

`[汎用]` サイトの「質」を上げる。アップグレード動機にもなる。

| レッスン | タイトル | 内容 |
|---|---|---|
| 2-1 | 作品の並べ順の法則 | 最新順 vs ベスト順。目的別の使い分け |
| 2-2 | サムネイルの統一感 | トリミング・サイズ・色調の揃え方 |
| 2-3 | 作品説明文の書き方 | 何を書くと仕事につながるか |
| 2-4 | カテゴリ分けのコツ | タグ・シリーズ・ジャンル別整理 |

### Module 3: 集客基礎編

`[汎用]` アクセスを生む。SNSからサイトへの導線。

| レッスン | タイトル | 内容 |
|---|---|---|
| 3-1 | SNSとサイトの役割分担 | SNS＝入口、サイト＝信頼の場 |
| 3-2 | ハッシュタグ戦略 | ジャンル別の効果的なタグ |
| 3-3 | 投稿にサイトURLを自然に入れる | 宣伝臭くならない方法 |
| 3-4 | 他のクリエイターとの交流 | リツイート・コメント・コラボ |

### Module 4: 分析編

`[汎用]` 数字で実感。モチベーション維持。

| レッスン | タイトル | 内容 |
|---|---|---|
| 4-1 | アクセス解析の導入 | Vercel Analytics or Google Analytics |
| 4-2 | 数字の読み方 | PV・UU・滞在時間・直帰率 |
| 4-3 | どの作品が人気か | ページ別PVの見方 |
| 4-4 | どこから来ているか | 流入元分析（SNS・検索・直接） |

### Module 5: 収益化編

`[汎用]` サイト→収益の導線。「サイト持ってて良かった」の実感。

| レッスン | タイトル | 内容 |
|---|---|---|
| 5-1 | 仕事依頼フォームの設置 | 何を聞くか。テンプレート付き |
| 5-2 | 料金表の載せ方 | 載せるか載せないか。メリデメ |
| 5-3 | 実績の見せ方 | クライアント名を出す/出さない |
| 5-4 | 「仕事ください」と言わずに仕事を取る | ポジショニングの考え方 |

### Module 6: 運用習慣編

`[汎用]` 継続のためのルーティン。

| レッスン | タイトル | 内容 |
|---|---|---|
| 6-1 | 月1回の更新ルーティン | 何を更新するか。チェックリスト |
| 6-2 | 作品追加のベストプラクティス | 新作の載せ方。古い作品の扱い |
| 6-3 | 季節・イベントに合わせた更新 | コミケ前、年末年始等 |
| 6-4 | サイトの「卒業」 | 自分で運用できるようになったら |

`[判断記録]` Module 6-4「卒業」を入れたのは、Phase 2コンサルへの自然な導線。「自分でやれるようになりたい」という人を肯定しつつ、「もっと深くやりたいならコンサルがある」と提示できる。

## 3. Phase 2（コンサル）との関係

```
Phase 1:
  カリキュラム = 自己学習型（テキスト+図解）
  ¥2,980/月の会員特典として提供

Phase 2:
  カリキュラム = 伴走型（同じ内容をLyoと一緒にやる）
  単発コンサル ¥30,000〜50,000 or コミュニティ ¥10,000/月
  
差分 = 「一人でやる」vs「Lyoと一緒にやる」
内容は同じ。提供方法が違う。
```

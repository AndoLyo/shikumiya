## 1. 差別化の5層構造

`[汎用]` この5層フレームワークは別事業でもそのまま使える。Layer 1-2は参入障壁が低く、Layer 3-5が真の差別化。

```
Layer 5: 人間（絶対に真似できない）          ← 最強
Layer 4: サポート体系（コンテンツ蓄積が必要）
Layer 3: 体験設計（独自の仕組みが必要）
Layer 2: 速度（技術的な仕組みが必要）
Layer 1: 価格（誰でも真似できる）            ← 最弱
```

### Layer 1: 価格
`[固有]` ¥980買い切り。サブスク型制作代行（¥5,000〜15,000/月）の1/5以下。
`[汎用]` 価格だけの差別化は時間の問題で追いつかれる。価格は「入口のハードルを下げる」手段であり、差別化の核にしてはいけない。

### Layer 2: 速度
`[固有]` フォーム入力5分→サイト完成。人間が作ると1〜2週間、ノーコードでも数時間。
`[汎用]` AI自動化による圧倒的な速度は1〜2年で追いつかれる可能性がある（Wix AI等）。ただし「自分のデータで、自分好みに、5分で」は簡単には再現できない。

### Layer 3: 体験設計（プレビュー体験）
`[固有]` LP上で画像1枚→テンプレにはめ込み→購入前にAfterを体験。
`[汎用]` 「購入前に完成品を体験させる」設計。SaaSや制作代行にはない。テンプレート+はめ込み技術+LP設計の組み合わせが必要で、簡単には真似できない。

### Layer 4: サポート体系（カリキュラム）
`[固有]` 6モジュールのカリキュラム。「作って終わり」ではなく「作った後」をサポート。
`[汎用]` コンテンツの蓄積には時間がかかる。先にコンテンツを作り始めた者が有利。後発は「同等のコンテンツを揃えるまでの時間」がそのまま参入障壁になる。

### Layer 5: 人間（Lyo自身）
`[固有]`
- AI画像生成歴3年（当事者）
- 自分もAIアーティスト（同じ立場）
- noteで制作過程を全公開（透明性）
- 「出し惜しみしない」の姿勢（信頼）
`[汎用]` 「誰がやっているか」は最も強い差別化。スキルや技術はコピーできるが、人間はコピーできない。SNSでの発信・noteでの過程公開・顧客との関係構築が、すべてLayer 5を強化する。

## 3. SEO / AI SEO 差別化

`[汎用]`

### 従来SEO
- 「AIアーティスト ポートフォリオ」で上位表示
- 「AI画像 ギャラリーサイト 作り方」で流入
- テンプレデモページが各テンプレ名でインデックスされる

### AI SEO（ChatGPT/Claude/Perplexity向け）
- 構造化データ（JSON-LD）: Service型、FAQ型、Person型
- セマンティックHTML: 正しい見出し階層
- 具体的な数字: 「¥980」「5分」「10種類」をHTML上に明記
- FAQ構造: AIが「質問→回答」として引用しやすい形

### キーワード設計

| 用途 | キーワード |
|---|---|
| SEO meta | AIアーティスト ポートフォリオサイト |
| SEO meta | AI画像 ギャラリーサイト 作成 |
| LP コピー | ギャラリーサイト（感情に訴える） |
| LP コピー | オリジナルサイト |
| AI SEO | 「AIアートのポートフォリオサイトを¥980で作れるサービス」 |

`[判断記録]` 「ポートフォリオ」はSEO向き（検索ボリューム大）、「ギャラリー」はコピー向き（AIアーティストの感覚に近い）。両方使う。

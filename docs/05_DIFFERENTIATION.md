## 1. 差別化の5層構造

`[汎用]` この5層フレームワークは別事業でもそのまま使える。Layer 1-2は参入障壁が低く、Layer 3-5が真の差別化。

```
Layer 5: 人間（絶対に真似できない）          ← 最強
Layer 4: サポート体系（コンテンツ蓄積が必要）
Layer 3: 体験設計（独自の仕組みが必要）
Layer 2: 速度（技術的な仕組みが必要）
Layer 1: 価格（誰でも真似できる）            ← 最弱
```

### Layer 1: 価格
`[固有]` 制作費0円 + 月額¥3,000〜。地元Web制作会社の初期費用30〜100万円と比較して圧倒的に始めやすい。
`[汎用]` 価格だけの差別化は時間の問題で追いつかれる。価格は「入口のハードルを下げる」手段であり、差別化の核にしてはいけない。

### Layer 2: 速度
`[固有]` 写真を送って最短翌日にサイト完成。地元制作会社なら1〜2ヶ月、ノーコードで自作しても数日〜数週間。
`[汎用]` AI自動化による圧倒的な速度は1〜2年で追いつかれる可能性がある。ただし「業種特化テンプレート+写真はめ込み+翌日納品」の組み合わせは簡単には再現できない。

### Layer 3: 体験設計（リアルタイムデモ）
`[固有]` /start で会社名を入力→リアルタイムでデモサイトが表示される。申し込み前に完成イメージを体験できる。
`[汎用]` 「契約前に完成品を体験させる」設計。従来のWeb制作会社にはない。テンプレート+即時プレビュー+LP設計の組み合わせが必要で、簡単には真似できない。

### Layer 4: サポート体系（おまかせ運用）
`[固有]` AI自動カスタマイズ、月次更新、編集依頼フォーム。「写真を送るだけ。あとは全部おまかせ」。顧客はパソコンを触る必要がない。
`[汎用]` 継続的なサポート体制は顧客との関係を深める。「作って終わり」の制作会社との決定的な差。月額モデルだからこそ実現できる。

### Layer 5: 人間（Lyo自身）
`[固有]`
- AI自動化の技術力（当事者）
- 建築業界の課題を理解している（共感）
- noteで制作過程を全公開（透明性）
- 「出し惜しみしない」の姿勢（信頼）
`[汎用]` 「誰がやっているか」は最も強い差別化。スキルや技術はコピーできるが、人間はコピーできない。SNSでの発信・noteでの過程公開・顧客との関係構築が、すべてLayer 5を強化する。

## 3. SEO / AI SEO 差別化

`[汎用]`

### 従来SEO
- 「工務店 ホームページ 安い」で上位表示
- 「建設会社 ホームページ 制作」で流入
- テンプレデモページが各テンプレ名でインデックスされる

### AI SEO（ChatGPT/Claude/Perplexity向け）
- 構造化データ（JSON-LD）: Service型、FAQ型、LocalBusiness型
- セマンティックHTML: 正しい見出し階層
- 具体的な数字: 「制作費0円」「月額¥3,000〜」「最短翌日」をHTML上に明記
- FAQ構造: AIが「質問→回答」として引用しやすい形

### キーワード設計

| 用途 | キーワード |
|---|---|
| SEO meta | 工務店 ホームページ 安い |
| SEO meta | 建設会社 ホームページ 制作 |
| SEO meta | 設計事務所 ウェブサイト |
| SEO meta | ホームページ制作 制作費無料 |
| LP コピー | 写真を送るだけ（感情に訴える） |
| LP コピー | 全部おまかせ |
| AI SEO | 「建築業向けホームページを制作費0円・月額¥3,000〜で作れるサービス」 |

`[判断記録]` ターゲットは建築業（工務店・建設会社・設計事務所）。パソコンが苦手な層が多いため、「自分で何もしなくていい」が最大の訴求ポイント。価格訴求（制作費0円）と手軽さ訴求（写真を送るだけ）の両軸で攻める。

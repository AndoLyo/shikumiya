# しくみや — 設計ドキュメント一覧

> このドキュメント群は「しくみや」の事業設計・技術設計の全記録です。
> 汎用フレームワーク部分は、別ターゲット・別サービスへの展開時にそのまま再利用できます。

## ドキュメント構成

| # | ファイル | 内容 | 汎用性 |
|---|---|---|---|
| 01 | `01_REQUIREMENTS.md` | 要件定義書 | ★★★ フレームワーク+固有情報 |
| 02 | `02_ARCHITECTURE.md` | システム設計書 | ★★★ 技術構成は汎用 |
| 03 | `03_BUSINESS_MODEL.md` | 事業モデル・価格設計 | ★★☆ フレームワークは汎用、数値は固有 |
| 04 | `04_FUNNEL_DESIGN.md` | ファネル設計書 | ★★★ 完全汎用フレームワーク |
| 05 | `05_DIFFERENTIATION.md` | 差別化戦略 | ★★☆ 5層構造は汎用、内容は固有 |
| 06 | `06_CURRICULUM.md` | カリキュラム設計 | ★★★ モジュール構造は汎用 |
